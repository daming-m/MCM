# 无人机信息管理系统（初期版本）— 需求文档（PRD）

## 1. 背景与目标
- 面向内部/小团队的无人机资产管理：支持无人机信息的录入、查询、删除、修改。
- 初期版本默认使用 SQLite，支持与 MySQL 自由切换。

## 2. 用户与使用场景
- 管理员登录系统后，维护无人机资产信息。

## 3. 功能范围
### 3.1 无人机信息管理（核心）
- 新增无人机
  - 用户只需输入基础信息（如名称/型号/序列号等）
  - 其余扩展属性由“AI 属性生成”自动补齐并保存
- 查询无人机（列表/按关键字搜索）
- 编辑无人机（基础信息可改；AI 生成属性支持重新生成）
- 删除无人机

### 3.2 登录与访问控制（必需）
- 未登录不可访问业务页面
- 登录失败提示

## 4. 数据与字段（初稿）
### 4.1 无人机基础字段（人工输入）
- 名称 name（必填）
- 型号 model（必填）
- 序列号 serialNo（必填，唯一）

### 4.2 AI 自动生成字段（系统生成）
- 最大速度 maxSpeedKmh
- 最大续航 flightTimeMin
- 最大载重 payloadKg
- 电池容量 batteryMah
- AI 评分 aiScore（0-100）

## 5. 非功能需求
- 可在 SQLite / MySQL 之间切换（通过配置 profile）
- 分层结构：controller / service / repository / domain
- 记录请求日志（拦截器打印请求信息）
